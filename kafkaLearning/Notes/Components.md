## Producer
* Source of Data
* Publish the Data or events

## Consumer
* Consumes the Data

## Broker 
* nothing but a server.
* Intermediate entity that helps in message exchanges between a producer and consumer.

## Cluster
* Group of Servers working for common purpose
* there can be one or more brokers in the kafka cluster.

## Topic
* Acts as table in kafka Eco system
* Topic specifies the classification or category of the message.
* Listeners can just respond to the messages that belong to the topics they are listening on.

## Partition
* Subdivision of each topic

## Offset
* Numbering of message

## Consumer Groups
* multiple consumers

## Zookeeper
* Keeps the track of kafka cluster
